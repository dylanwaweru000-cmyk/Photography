# Photography by Dylan

A responsive photography portfolio website showcasing landscape and travel photography

## About

This project is a personal photography gallery site featuring:
- A home page with an introductory hero image and quote
- An About section describing my journey into photography
- A gallery of landscape and travel photos with captions
- A contacts section for reaching out

## Tech Stack

- HTML5
- CSS3 (Flexbox for layout and centering)
- Git & GitHub for version control

## Features

- Fully responsive image gallery
- Styled photo cards with borders and captions
- Centered navigation and content sections
- Clickable email link in Contacts
